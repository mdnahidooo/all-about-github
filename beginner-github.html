Beginner GitHub Instruction

…or create a new repository on the command line

  1. echo "# all-about-github" >> README.md
  2. git init
  3. git add .
  4. git commit -m "Commit will be related to your work or update"
  5. git branch -M main
  6. git remote add origin https://github.com/mdnahidooo/all-about-github.git
  7. git push -u origin main

…or updated a repository by the command line

  1. git add .
  2. git commit -m "Commit will be related to your work or update or changes to your code"
  3. git push

#…or push an existing repository from the command line

  1. git remote add origin https://github.com/mdnahidooo/all-about-github.git
  2. git branch -M main
  3. git push -u origin main
